# Tesla Wrapper

**Play it live:** https://tesla-wrap-viewer.onrender.com  
Multiplayer wrap sandbox for the Tesla Model 3: drive together, preview wraps in real time, chat, and copy each other’s designs instantly.

## Features
- Real-time multiplayer driving with wrap/color syncing
- Custom wrap upload + presets
- Direct player chat and wrap copying
- Day/Night lighting (auto by local sunrise/sunset; manual toggle)
- Persistent player name and wrap across reloads

## Roadmap
- New Tesla 3D models
- Expand the city without killing performance (streaming/tiling, LODs, impostors)

## Contribute
- Developers: jump in via issues/PRs—help with world streaming, perf, and new gameplay ideas.
- Designers/3D artists: contribute new Tesla models in Blender and fresh wrap templates.
